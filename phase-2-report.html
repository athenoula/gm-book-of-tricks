II. The Features

Everything you asked about — assessed for impact, effort, and cost.

Feature Spotlight

PDF Upload & Viewer High Priority

Upload rulebooks, modules, and reference PDFs. View them in-browser with search, bookmarks, and annotation overlays. Most GM tools handle PDFs poorly — this is a real gap.

Recommended Stack

react-pdf (by Wojtek Maj) for simple viewing — 1M+ weekly npm downloads, wraps Mozilla's pdf.js. If you want a full toolbar with search and thumbnails, step up to @react-pdf-viewer/core with its plugin system.

Annotations stored as JSON overlays in Supabase (page number, coordinates, text) — rendered on top of the PDF canvas. No need for expensive commercial SDKs like PSPDFKit.

Cost

$0 — react-pdf is open source. Storage via Supabase free tier (1GB). Keep PDFs under 10-20MB for good UX.

Feature Spotlight

Character Image Upload High Priority

Upload portraits for PCs, NPCs, monsters, and locations. Display in circular ornamental frames throughout the app. Build a visual library tied to your campaign entities.

Implementation

Compress in-browser before upload using browser-image-compression (target 1MB max, 1920px max dimension). Store in Supabase Storage. Serve with Supabase's built-in CDN.

For the gallery: react-photo-album (responsive grid) + yet-another-react-lightbox (zoom/navigate). Modern, lightweight, composable.

III. The Communications Hub

Send handouts, recaps, and reminders without leaving your console.

This is the killer differentiator. Right now every DM manually copies content from their prep tool to Discord/WhatsApp/email. We can automate that entire flow.

IV. The Visual Elevation

Making it feel like a fantasy game — not a SaaS dashboard. Everything you see on this page is achievable.

The Warm Craft aesthetic from Phase 1 (dark charcoal + amber + serif headings) is the foundation. Phase 2 layers on illustrated, physical, tangible elements that make the tool feel like an artefact you'd find in a wizard's study.

This very report is the demo. Every visual technique used here — the ornamental dividers, gold foil text, parchment cards, floating embers, vignette, noise texture, flickering torch glow — is CSS-only, zero dependencies, zero cost.

More Visual Techniques (All Free)

Vignette Overlay

One ::after pseudo-element with a radial gradient. You're looking through it right now. Instant depth.

Paper Grain Noise

Inline SVG <feTurbulence> as a background-image at 2-3% opacity. Adds tactile quality to every surface.

Inline SVG components — diamond, dot, line. 5 reusable components transform the feel of every page.

CSS @keyframes varying opacity and brightness. 🔥 Apply to key UI icons for that candlelit vibe.

Aged Image Filter

V. Maps & Handouts

Interactive world maps, shareable handouts, and fog of war.

Interactive Campaign Map Medium Effort

Inspired by LegendKeeper's Atlas — the most praised feature in the worldbuilding tool space. Upload a world/region map, place clickable pins that link to locations in your campaign. Nested maps (click a city → zoom into city map).

Built with Leaflet.js (40KB, free) using custom image tiles. Pins link to your existing location entities. SVG paths connect locations (dotted = unexplored, solid = travelled).

Shareable Handout Content

Any timeline block, NPC card, location entry, or image can be packaged as a shareable handout. One click sends it to Discord (embed), WhatsApp (image + text), or email (styled HTML). This is the "send this to my players" flow — the thing no other tool does well.

The handout system is the glue. PDF pages become handouts. Map regions become handouts. Session notes become recaps. The communications hub sends them. One unified flow.
